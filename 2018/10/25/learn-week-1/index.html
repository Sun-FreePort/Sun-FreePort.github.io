学习周记（#1）

PHP

本周计划：每日一个函数。

array_change_key_case

将所有键名修改为全大写或小写

类型 值 注释
参数值 array 目标数组
参数值 int 转化目标，值为：CASE_LOWER/CASE_UPPER
返回值 array/bool 传入值非数组时，返回 null

array_chunk

将数组按数量分割为多个新数组

类型 值 注释
参数值 array 目标数组
参数值 int 新数组长度
参数值 bool 保留数组键名，默认为 false
返回值 array/bool 传入值非数组时，返回 null

array_column

获取多维数组中指定列

类型 值 注释
参数值 array 目标数组
参数值 mixed 目标列（数字下标、键、属性名等，为 null 将返回整个数组）
参数值 mixed 另一目标列，将该列的值设为键，默认为 null
返回值 array/bool 传入值非数组时，返回 null

array_combine

使用两个数组作为键值对，创建新数组

类型 值 注释
参数值 array 键名数组
参数值 array 键值数组
返回值 array/bool 两数组长度不同，返回 null
